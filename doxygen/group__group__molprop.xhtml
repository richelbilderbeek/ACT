Generates a LaTeX table containing the statistics (RMSD from experiment) of a calculated property per molecule category.

Parameters
[in]msg_handlerFor debugging info
[out]fpFile pointer to write to
[in]epropThe property of choice
[in]mpArray of molecules
[in]qmcQM calc statistics
[in]outlierOutlier indicates a level (in units of sigma, one standard deviation). Calculations that deviate more than this level from the experiment are not taken into account when computing statistics. Moreover, the outliers are printed to the standard error. If outlier is 0, no action is taken.
[in]cListStructure containing all categories and the number of molecules per category
[in]gmsStructure containing selections of which molecules to output
[in]imsThe actual selection of the right set
