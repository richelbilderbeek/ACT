Compute and return the Boltzmann factor Takes into account both local annealing (within a generation) and global annealing (spreading over generations) depending on user specified command line options. Will return 1/temperature that is, not multiply with the Boltzmann constant. This is because the deviation used in the MCMC algorithm is not an energy anyway.
