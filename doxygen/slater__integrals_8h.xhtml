Description

Provides routines for computing Coulomb integrals analytically. The Slater based code depends on the optional CLN library for arbitrary precision arithmetic.

Author
David van der Spoel david.nosp@m..van.nosp@m.dersp.nosp@m.oel@.nosp@m.icm.u.nosp@m.u.se
